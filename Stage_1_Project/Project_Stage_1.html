lesson 1: Basics of the Web and HTML

How does the web work?
  • The web is a bunch of computers that communicate with each other
  • When a person goes to a web page, their computer sends a HTTP request to a server
  • Server: Computer specialized in storing websites rather than displaying it to users directly
  • Browser: Front end that let users experience websites written in HTML
  • HTTP: The protocol used to exchange information between different computers
  • Information is not displayed the way it is organized in the editor. A line break in the editor does not mean equal a line break on the displayed website.
  • Because Computers are "stupid", they only understand correct code and are not able to correct mistakes.
What is HTML?
  • HTML stands for Hyptertext Markup Language.
  • HTML documents form the majority of the content on the web.
  • HTML documents contain text content which describes "what you see" and markup which describes "how it looks"
  • The tree like structure comes from the fact that HTML elements can have other elements inside of them
  • The Document Object Model (DOM) is an programming interface for valid HTML documents.
  • It defines the logical structure of documents and the way a document is accessed and manipulated.
What are Tags and Elements?
  • Tags always have a Opening Tag and an Ending Tag
  • Are not displayed on the webpage and are only aimed to the computer
  • Can be Inline or Block
  • Can have attributes, e.g. the img Tag for pictures has the attribute of a web adress
Inline vs Block Elements
  • HTML Elements are either inline or block.
  • Block elements form an "invisible box" around the content inside of them
